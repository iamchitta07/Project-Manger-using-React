# refs-demo-project

A small starter React project using Vite and Tailwind CSS.

## Prerequisites
- For npm/yarn: Node.js (recommended >=16)
- For Bun: Bun (https://bun.sh/) if you prefer using Bun as runtime/package manager

## Install & Run

### Using Bun
Install dependencies and start the dev server:

```bash
bun install
bun run dev
```

Build and preview:

```bash
bun run build
bun run preview
```

### Using npm
Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

Build and preview:

```bash
npm run build
npm run preview
```

### Using Yarn
Install dependencies and start the dev server:

```bash
yarn
yarn dev
```

Build and preview:

```bash
yarn build
yarn preview
```

## Available scripts
The project scripts are defined in [package.json](package.json):

- `dev`: start the Vite development server
- `build`: build for production
- `preview`: locally preview the production build
- `lint`: run ESLint

## Files
- Main entry: [src/main.jsx](src/main.jsx)
